# Panel de Clientes - Trabajo Integrador

Este proyecto es el Trabajo Práctico Integrador Final para la cátedra de Programación Visual.

**Descripción del Proyecto:**
Consiste en una aplicacion web SPA desarrollada con React y Vite, diseñada para la gestion de un directorio de clientes. La plataforma se conecta de forma asincronica a la FakeStore API y cuenta con un sistema de autenticacion de usuarios administrado a traves de Context API y LocalStorage, permitiendo control de accesos basado en roles (Soporte y Gerencia). 

Entre sus funcionalidades principales destacan: la visualizacion del listado de clientes con un motor de busqueda en tiempo real, validacion estricta de formularios para altas simuladas, enrutamiento dinamico para la visualizacion profunda de fichas individuales y renderizado condicional de acciones criticas segun los permisos del administrador
